Babel configuration
Command: npm install --save-dev babel-plugin-module-resolver
URL: https://www.npmjs.com/package/babel-plugin-module-resolver

Google Fonts
Command: expo install expo-font @expo-google-fonts/roboto      
UR: https://docs.expo.io/guides/using-custom-fonts/

Native Base UI
Command: npm install native-base
Command: expo install react-native-svg@12.1.1
Command: expo install react-native-safe-area-context@3.3.2
URL: https://docs.nativebase.io/install-expo

SVG transformer
Command: npm install --save-dev react-native-svg-transformer
URL: https://github.com/kristerkari/react-native-svg-transformer
(Remember to create/config the new file metro.config.js + add svg to @types)

React Navigation 
Command: npm install @react-navigation/native
Command: expo install react-native-screens react-native-safe-area-context 
Command: npm install @react-navigation/native-stack
Command: npm install @react-navigation/bottom-tabs

Expo Image Picker
Command: npx expo install expo-image-picker
URL: https://docs.expo.dev/versions/latest/sdk/imagepicker/
(lembrar de adicionar as configs no app.json)

Expo File System
Command: npx expo install expo-file-system
URL: https://docs.expo.dev/versions/v47.0.0/sdk/filesystem/

React Hook Form + Yup
Command: npm install react-hook-form
Command: npm install @hookform/resolvers yup
URL: https://react-hook-form.com/get-started/

Axios 
Command: npm install axios
URL: https://www.npmjs.com/package/axios

Expo Async Storage
Command: expo install @react-native-async-storage/async-storage
URL: https://docs.expo.dev/versions/latest/sdk/async-storage/
